# 🌦 SkyPulse — Weather Web Application

SkyPulse is a modern weather web application that allows users to view real-time weather information, explore an interactive weather map, and manage search history using the OpenWeatherMap API.

---

## 🚀 Features

- 🌍 Live Weather Dashboard
- 🗺 Interactive Weather Map (Leaflet.js)
- 📍 Select City Directly from Map
- 🕘 Search History Tracking
- 🔎 City Weather Search
- 💾 LocalStorage Data Saving
- 🎨 Modern Responsive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Leaflet.js
- OpenWeatherMap API
- Font Awesome
- LocalStorage
